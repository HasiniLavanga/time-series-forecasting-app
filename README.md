# Time Series Forecasting App

A Streamlit-based machine learning application for time series data visualization and forecasting.

## Features
- CSV File Upload
- Interactive Time Series Visualization
- Dataset Preview
- Date and Target Column Selection

## Tech Stack
- Python
- Streamlit
- Pandas
- Plotly

## Dataset
Daily Climate Time Series Dataset

## Project Structure

time-series-forecasting-app/
│
├── app.py
├── requirements.txt
├── README.md
└── sample_data.csv

## Run Locally

```bash
streamlit run app.py
```
